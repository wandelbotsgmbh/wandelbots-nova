# Migration guide v1 -> v2

This migration guide highlights the changes in the Wandelbots NOVA API v2 compared to the Wandelbots NOVA API v1.
v2 comes with a lot of improvements like a less complex API, a more intuitive way to interact with the robot,
and a more robust and reliable communication between the robot and the IPC.

Read through the [NOVA API v2 quickstart](/25.3/api/v2-quickstart) and try it out using the [NOVA API v2 reference](https://portal.wandelbots.io/docs/api/v2/ui/#/)!
If you have any questions or need help, feel free to ask in the Wandelbots Robotics community.

Wandelbots NOVA API V1 will be **deprecated on xx/xx/xx**.

Refer to this table for the parameter conversion between v1 and v2:
